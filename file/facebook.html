Facebook ចេញ​មុខងារ​ថ្មី​ដាក់​រូបភាព​ Profile ជាប់​​ឥន្ទធនូ​អបអរ​ច្បាប់​រៀបការ​ភេទ​ដូច​គ្នា​នៅ​អាមេរិក
27 Jun, 2015 09:59

​ក្រុមហ៊ុន​បណ្ដាញ​សង្គម​ Facebook ​បាន​​ដាក់​ឲ្យ​ប្រើ​ប្រាស់​នូវ​មុខងារ​រូបភាព​ថ្មី​​សម្រាប់​អបអរ​ទៅ​ដល់​សេចក្ដី​សម្រេច​របស់​តុលាការ​កំពូល​​អាមេរិក​ដែល​បាន​ទទួល​ស្គាល់​ និង​​​អនុញ្ញាត​ឲ្យ​មនុស្ស​ភេទ​ដូច​គ្នា​អាច​រៀប​ការ​ជា​មួយ​គ្នា​បាន។

​នៅ​ថ្ងៃ​ទី​ ២៦ ខែ មិថុនា​ម្សិល​មិញ​នេះ​ តុលាការ​កំពូល​របស់​អាមេរិក​បាន​សម្រេច​ចិត្ត​​ និង​​យល់​ព្រម​ឲ្យ​​​​មនុស្ស​មាន​​ភេទ​ដូច​គ្នាអាច​រៀប​ការ​ជា​មួយ​គ្នា​បាន​នៅ​​ក្នុង​ទឹក​ដី​អាមេរិក​។ ដូច្នេះ​ដើម្បី​អបអរ​ដល់​សេចក្ដី​សម្រេច​នេះ​ Facebook ក៏​​បាន​បង្កើត​ Tool មួយ​ដែល​អនុញ្ញាត​ឲ្យ​អ្នក​ប្រើ​ប្រាស់​​បង្កើត​រូបភាព Profile ​ដែល​មាន​រូបភាព​ជា​ឥន្ទធនូ​ដ៏​ស្រស់​ស្អាត​ជាប់​ជា​មួយ​នឹង​រូប​ Profile ។ មិន​មែន​មាន​​តែ​ Facebook តែ​មួយ​ប៉ុណ្ណោះ​នោះ​ទេ​​ ក្រុមហ៊ុន​ផ្សេង​ទៀ​ត​ដូច​ជា​ Apple, Google, Uber និង Salesforce ក៏​បាន​អបអរ​ទៅ​នឹង​សេចក្ដីសម្រេច​នេះ​ដែរ។

ចំពោះ​អ្នក​លេង​ Facebook ផ្សេងៗ​​​ក៏​អាច​ធ្វើ​តេស្ដ​សាក​ល្បង​​នូវ​ Tool ​ថ្មី​នេះ​បាន​ដែរ ​ដោយ​គ្រាន់​តែ ​ចុច​នៅ​ទីនេះ ឬ ចូល​ទៅ​កាន់​វិបសាយ (https://www.facebook.com/celebratepride)។ តែ​ត្រូវ​ចាំ​ថា​ អ្នក​បាន​ Log in ចូល​ទៅ​ក្នុង​គណនី​របស់​អ្នក​។ បន្ទាប់​មក​ចុច​លើ​ប៊ូតុង​ “Use as Profile Picture” នោះ​អ្នក​នឹង​បាន​រូបភាព​ Profile ដែល​មាន​រូប​ឥន្ទធនូ​ជាប់​ជា​មួយ៕
